# VibeXthon-26 Hackathon Portal

Official hackathon management portal for VibeXthon-26.

## Features

- Secure Team Registration
- Email Verification
- Unique Team Name Enforcement
- Role-Based Authentication
- Project Submission System
- Judge Panel
- Admin Panel
- Submission Deadline Control

## Tech Stack

Frontend:
- Next.js
- Tailwind CSS

Backend:
- Firebase Authentication
- Firestore Database

Hosting:
- Vercel

## User Roles

### Participant
- Register Team
- Submit Project (Once)
- View Dashboard

### Judge
- View Submissions
- Provide Feedback
- Select / Reject Teams

### Admin
- Manage Submission Window
- Control Judge Access
- View & Delete Data

## Deployment

1. Install dependencies

2. Run development server 
3. Configure Firebase credentials in `.env.local`
